# Molham Sakkal — Academic Profile

Personal academic website for **Molham Sakkal**, Laboratory & Research Supervisor at the Health and Biomedical Research Center (HBRC), Al Ain University, UAE.

## About

This site showcases Molham's research in cancer cell biology, drug delivery systems, molecular oncology, and nanoformulations. Built as a fork of [ScholarSite_2.0](https://github.com/muhammedrashidx/ScholarSite_2.0).

## Live Site

🔗 [scholarlybrightminds.github.io/molhamsakkal](https://scholarlybrightminds.github.io/molhamsakkal/)

## Pages

- **Home** — Bio, social links, and overview
- **About** — Education and career background
- **Ongoing Research** — Current research projects
- **Publications** — Auto-rendered from Google Scholar / Scopus data

## Tech

Static HTML/CSS/JS hosted on GitHub Pages. Publications are rendered client-side from JSON data in `data/serpapi/`.
